## 파일 이름 규칙

파일 이름은 실제 면접에서 나올 수 있는 형태를 영어 소문자와 하이픈을 사용해 작성합니다.

```test
process-vs-thread.md
tcp-vs-udp.md
database-index.md
b-tree.md
deadlock.md
http1-http2-http3.md
handler-looper-messagequeue.md
```

피해야 할 예시는 다음과 같습니다.

```text
ProcessVsThread.md
프로세스와스레드.md
TCP_UDP.md
정리.md
새문서.md
```

## 출처 작성 규칙

모든 문서는 하단에 `참고 자료` 섹션을 포함하는 것을 원칙으로 합니다. 출처를 작성할 때는 다음을 지켜주세요.

- 단순히 링크만 나열하지 말고, 실제로 참고한 자료만 작성합니다.
- 블로그 글을 참고한 경우, 가능하면 공식 문서나 표준 문서도 함께 확인합니다.
- 긴 문장을 그대로 복사하지 않습니다.
- AI 도구를 사용해 초안을 작성한 경우에도, 최종 내용은 반드시 신뢰할 수 있는 자료로 검증합니다.
- 출처가 불명확한 내용은 단정적으로 작성하지 않습니다.

## 기여하기

이 저장소는 질문을 중심으로 CS 지식을 함께 정리해 나가는 아카이브입니다.

잘못된 내용 수정, 부족한 설명 보완, 새로운 질문 추가, 꼬리 질문 확장 등 모든 형태의 기여를 환영합니다.

내용추가의 경우 각 질문과 파일이 1대1 대응하며, 항상 모든 자료의 출처가 적혀있어야합니다.
