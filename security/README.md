# Security

인증, 인가, 웹 취약점, 개인정보 보호와 관련된 면접 질문을 정리합니다.

## 질문 목록

- 인증과 인가의 차이는 무엇인가요?
- 세션 기반 인증과 토큰 기반 인증의 차이는 무엇인가요?
- JWT의 장단점은 무엇인가요?
- SQL Injection이란 무엇인가요?
- XSS란 무엇인가요?
- CSRF란 무엇인가요?
- 비밀번호는 어떻게 저장해야 하나요?
- 다중 서버 환경에서 세션 기반 인증 방식을 사용하는 경우 발생할 수 있는 문제점은 무엇인가요?
- 대칭키 및 비대칭키 암호화 방식에 대해 설명해주세요.
- CSRF 공격에 대해 설명해주세요.
- JWT 특징과 주의 사항을 설명해주세요.
- Spring OAuth2 소셜 로그인에 대해 설명해주세요.
- Spring Security JWT 인증 구현에 대해 설명해주세요.
- Spring Security의 인증과 인가에 대해 설명해주세요.
- Spring SecurityFilterChain 구조에 대해 설명해주세요.
- SQL 인젝션에 대해 설명해주세요.
