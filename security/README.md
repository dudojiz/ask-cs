# Security

인증, 인가, 웹 취약점, 개인정보 보호와 관련된 면접 질문을 정리합니다.

우선 작성할 질문:

- 인증과 인가의 차이는 무엇인가요?
- 세션 기반 인증과 토큰 기반 인증의 차이는 무엇인가요?
- JWT의 장단점은 무엇인가요?
- SQL Injection이란 무엇인가요?
- XSS란 무엇인가요?
- CSRF란 무엇인가요?
- 비밀번호는 어떻게 저장해야 하나요?
